# Data Notes

This repository includes the small metadata files, derived statistics, notebooks, challenge description files, and Tableau workbook needed to understand the analysis.

## Included

- `2018 MC1.pdf`: VAST Challenge 2018 Mini-Challenge 1 prompt.
- `Data Description.docx`: description of the provided challenge data.
- `AllBirdsv4.csv`: original verified bird metadata.
- `updated_AllBirdsv4.xlsx`: cleaned bird metadata used in this project.
- `Test Birds Location.csv`: locations of Kasios's claimed Pipit recordings.
- `Lekagul Roadways 2018.bmp`: map used as the spatial background.
- `ridma_tableau.twbx`: Tableau workbook with the spatial-temporal dashboards.
- `pipit_spectral_statistics.csv`: derived spectral features for verified Pipit recordings.
- `kasios_spectral_statistics.csv`: derived spectral features for Kasios recordings.

## Excluded

The raw and generated media folders are intentionally excluded from GitHub because they are large:

```text
all_birds/
kasios_birds/
spectral_plots/
```

The notebooks expect the raw audio folders to exist at the project root with these exact names:

```text
all_birds/
kasios_birds/
```

The `spectral_plots/` folder can be regenerated by running the notebooks.

## Coordinate Note

The challenge data description lists the alleged dumping site as `(148,40)`. In this project, the Tableau analysis indicates the working dumping-site location near `(148,160)`. The README documents this as a correction for the map interpretation used in the dashboards.

## Reproducibility Note

The included derived statistics CSV files allow the spectral comparison notebook to communicate the core result without requiring GitHub to store the full MP3 dataset.
